# WorldQuest 🌍

WorldQuest is a gamified educational adventure platform that transforms learning into an exciting journey. Players create their own explorer character and travel through story-driven chapters filled with world knowledge, quizzes, puzzles, achievements, and progression systems.

## ✨ Features

- **Authentication**: Secure sign-up and login with Firebase.
- **Character Creation**: Choose your explorer name and unique avatar.
- **Adventure Mode**: 5 narrative-driven chapters exploring history, geography, and culture.
- **Quiz System**: 45+ educational questions with high-stakes boss challenges.
- **Puzzle Vault**: 30+ interactive sliding puzzles to solve and replay.
- **XP & Level System**: Earn XP, level up, and unlock prestigious titles from "Explorer" to "World Scholar".
- **Achievement System**: Unlock milestones like "First Steps" and "History Buff".
- **Daily Streak System**: Maintain your momentum and earn bonus XP for consecutive play.
- **Global Leaderboard**: Compete with other explorers around the world.
- **Explorer Profile**: A comprehensive dashboard showing your journey and accomplishments.

## 🛠️ Tech Stack

- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS.
- **Backend**: Firebase (Authentication, Firestore).
- **Styling**: Modern dark-mode UI with custom animations and glassmorphism.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- Firebase Project

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env.local` file based on `.env.local.example` and add your Firebase credentials.
4. Run the development server:
   ```bash
   npm run dev
   ```

## 📜 Story Premise

The World Archive has lost five Knowledge Crystals that preserve humanity's knowledge. As an explorer, your mission is to recover these crystals by completing challenges across the globe. Each completed chapter restores one Knowledge Crystal. Become a World Scholar and restore the World Archive!
